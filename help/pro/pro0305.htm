进入项目卡片，点击"任务"按钮，进入以下界面：

 

点击"人力使用"按钮，进入以下界面：

 

这是整个项目人员的工作安排的一览表。下表是对相关字段的解释信息：

字段 描述信息
名称 列出人员和人员所负责的各任务。如果人员负责的任务下面还有子任务，则所有下属子任务也全部列出
工作流/文档 与此任务相关的工作流和文档。点击可以进入查看具体的信息
负责人 任务（包括各级子任务）的负责人
完成 任务已完成的百分比
工期 任务（包括各级子任务）的工期
起始日期 任务（包括各级子任务）的起始日期
结束日期 任务（包括各级子任务）的结束日期
