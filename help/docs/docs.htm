在线手册->知识文档管理 后退 前进 

知识文档管理模块为存储和编辑各种不同类型的企业文档提供了一个平台，核心是一个中心数据库，用户可以方便的通过浏览器进行文档的搜索和读取。本模块提供了以下主要功能：

  • 编辑、存储和发布各种类型的文档
  • 文档的有序管理
  • 维护和执行讨论会
  • 定义新闻、文档和邮件模板
  • 建立并维护内部和外部网站
  • 自动生成相关的统计报告
  • 相关日志
  • 自定义功能


所有的文档都可以设置相关的安全选项，只有具备相当安全级别或特定角色的人，才可以对文档进行操作（查看、编辑、删除等）。另外，可以将文档与相关的项目、客户、人力资源、资产产品等进行链接，使各模块间的信息完全融合在一起。
