在线手册->知识文档管理->主目录设置 后退 前进 

建立主目录时，进入文档模块点击"维护、基本设置、主目录/分目录/子目录"中，出现相关的系统已建立的目录列表，见如下界面：

（图甲）

 

用户可以打开现有的目录列表进行编辑修改，即选中目标主目录打开，见如下界面：

 

若创建新的目录机构，则在图甲中点击"新建"，出现一下界面：

 

下表是对相关字段的属性的解释：

字段

解释信息

基本信息

说明

对主目录名称的定义

顺序

该主目录在主目录列表中的排序列号

图片

为建立的主目录加入一张图片，这里选择的图片将成为主目录的默认图片。

按"S-保存"保存该设置，如需建立其他的主目录可重复以上操作。
