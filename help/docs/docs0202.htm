以下是对各功能按钮的解释信息

 

功能按钮

描述信息

保存

按此功能按钮保存文档，并按照选择的发布类型发布到相关文档目录或新闻页中。

草稿

按此按钮将所编辑的文档保存为草稿副本，但不发布。

预览

按此功能按钮可以对文档进行预览，同时被编辑的文档被保存为草稿。

HTML

按此功能按钮将对文件进行HTML格式下的编辑。

页眉

按此功能按钮将在显示或隐藏文档的页眉信息之间切换。
