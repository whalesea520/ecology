一、环境要求

CMS采用当前最先进的Web编程语言JAVA，遵循J2EE标准。对于操作系统，可以支持Windows98以上版本的微软平台，支持Linux，Solaris等基于UNIX的平台，具有非常优越的跨平台性能；对于数据库，支持SQL-SERVER，ORACLE。CMS采用的B/S架构，是当前运用最广泛的第三代瘦客户端技术，客户端只需要安装IE浏览器，就可以操作CMS，而服务器端只要有Web服务器（如Windows自带的IIS）和JVM(JAVA虚拟机)，就可以运行CMS，因此CMS对于环境的最低要求是很低的，用户对于运行环境的选择是非常灵活的。

基于上述特性，CMS对环境的要求如下：

①．硬件：

服务器端：X86 PC，256M内存，800M硬盘空间，10M网卡

客户端：10M网卡

②．软件：

服务器端：Window98 以上，Linux ，Solaris ，AIX等支持JAVA的操作系统，SQL SERVER或者Oracle，一种Web Server，JAVA JVM ，JAVA Application Server

客户端：IE浏览器

③．网络：

64k带宽，10M网卡

二、数据库服务器

在安装CMS前，需要先安装好数据库。

1、 安装SQL-SERVER2000

①．SQL-SERVER2000的安装及管理请参照软件自带的帮助文档。

②．将SQL-SERVER2000安装为可交互式登录模式，设置数据库管理员sa的密码，假定为 123456。

③．创建一个数据库，假定为CMS。

2、 安装ORACLE

 

①．ORACLE的安装及管理请参照软件自带的帮助文档。

②．在数据库服务器端使用ORACLE数据库安装向导创建一个数据库，假定为CMS，DBA帐号CMS，密码CMS

 

 三、系统安装

(1)、自动安装

 自动安装是专门针对WIN2000平台的快速安装方式。

 安装程序setup.exe打包了CMS系统、JDK(JAVA RUNTIME)、Resin(APPLICATION SERVER)、Apache(WEB SERVER)等软件。

只要双击运行setup.exe，安装向导将自动引导你一步步完成CMS应用系统及系统运行环境的安装配置。 

默认安装路径为C:\ZARVASOFT。 

重要提示：因为Apache是第三方免费软件，安装程序只是调用启动Apache安装，必须将Apache的安装路径设定为CMS系统安装路径(如默认安装路进为C:\ZARVASOFT)，否则需要手动配置Apache的httpd.conf文件。

 安装玩请重启电脑。

(2)、手动安装

需要安装CMS，Web Server，JVM和JDK，JAVA Application Server。现在以Windows 2000操作系统，Apache(WEB SERVER)，JDK 1.3.1 ，Resin Application Server 1.2.1为例，进行安装说明。假定安装路径为C:\ZARVASOFT。

1、 安装JVM 和 JDK 1.3.1

①．将jdk1.3.1.rar解压缩到C:\ZARVASOFT。

②．右击我的电脑，选择属性，选择高级，点击环境变量，将JAVA虚拟机（JVM）的运行路径加入环境变量 PATH 中。按上述假定因为 C:\ZARVASOFT\jdk1.3.1\bin

③．打开DOS窗口，在任意路径执行 java 命令 ， 以监测JAVA虚拟机（JVM）的运行路径是否正常。

2、 安装Resin Application Server 1.2.1

①．将resin1.2.2.rar解压缩到C:\ZARVASOFT。

②．打开DOS窗口，进入C:\ZARVASOFT\resin1.2.1\bin，如果解压缩的目录为其它目录，请进入相应目录，运行 srun.exe -install 将resin 安装为Windows的后台自启动服务（从服务中启动Resin）

③．配置Resin，用记事本或其它文本编辑工具打开C:\ZARVASOFT\resin1.2.1\conf\resin.conf ，将所有C:\ZARVASOFT的地方改为实际的系统安装路进，如安装路径为C:\ZARVASOFT，则可略过这一步。


3、 安装 Apache

①．运行ApacheGroup\apache_1.3.26-win32-x86-no_src.exe

②．根据安装向导提示一步步进行Apache安装，建议将安装路径设置为C:\ZARVASOFT。

③．配置Apache，先将ApacheGroup\conf\hppd.conf拷贝覆盖到C:\ZARVASOFT\Apache\conf\httpd.conf，用记事本或其它文本编辑工具打开C:\ZARVASOFT\Apache\conf\httpd.conf ，将所有C:\ZARVASOFT的地方改为实际的系统安装路进，如安装路径为C:\ZARVASOFT，则可略过这一步。


4、 安装CMS

①．将CMS.rar解压缩到C:\ZARVASOFT 即可。

5、重启电脑


(3)、数据库初始化

在IE浏览器中访问  http://IP地址，将出现数据库设置页面。

1、请根据实际情况在该页面填入下列信息：

 

数据库类型

数据库服务器IP

数据库名称

用户名

密码

使用现有数据库

 

2． 点击创建按钮当提示信息 success 出现后，数据库已成功创建并配置。

3．从服务中重启Resin Servlet Runner

(4)、运行系统

现在，安装已经全部完成，你可以登录CMS，使用其强大的协同商务功能了！

CMS登录方式，在IE浏览器中访问 http://IP地址

CMS 的初始用户为 weaveradmin 密码为 1


四、系统卸载

1.        卸载Resin服务：打开DOS窗口，进入C:\ZARVASOFT\resin1.2.1\bin，运行 srun.exe -remove 将Resin Servlet Runner从NT服务中卸除。

2.        卸载Apache：打开“控制面板”“添加/删除程序”，将Apache删除。

3.        卸载CMS应用系统：打开“控制面板”“添加/删除程序”，将CMS删除。

4.        删除C:\ZARVASOFT目录。

5.        重启电脑。

五、问题

6.        如果出现找不到服务器或其它失败页面，请检查Apache的配置，Apache服务是否启动（在服务中）。

7.        如果出现 Can't contact resin at 6802 port ，请检查Resin的配置，并确认JVM 是否安装成功（在DOS窗口敲入java 命令检查），Resin服务是否启动（在服务中） 。

8.        如果CMS创建数据库失败，请检查数据库是否启动，用户名密码是否正确。失败页面会给出相应失败提示
